Analysis

The analysis module can help you to play with dex/class format, with more high level information.

The first thing to do is to open your file and call the main analysis class : 

a = androguard.AndroguardS( "path_to_your_file" )

# VMAnalysis class takes the virtual machine as first input (e.g : DalvikVMFormat, JVMFormat)
x = analysis.VMAnalysis( a.get_vm() )

Next, you can iterate over each method to get the corresponding analysed method, to do that you have an access to the get_method method where the parameter is a method object : 

for method in a.get_methods() :
   ### g is the analyzed method
   g = x.get_method( method )

   ## print few information
   print method.get_class_name(), method.get_name(), method.get_descriptor(), method.get_code().get_length(), method.get_code().registers_size.get_value()

If you use the androlyze tool, you can used the AnalyzeAPK or AnalyzeDex directly : 

a, d, dx = AnalyzeAPK("test.apk") # a is an APK object, d is a DalvikVMFormat object , and dx is a VMAnalysis object

d, dx = AnalyzeDex("test.dex") # d is a DalvikVMFormat object , and dx is a VMAnalysis object

CFG

In the analysed method, you have a field basic_blocks which represents all the basic blocks in the method. You can used the get() method of this object to iterate over all basic blocks : 

for method in a.get_methods() :
   g = x.get_method( method )

   idx = 0
   for i in g.basic_blocks.get() :
      print "\t %s %x %x" % (i.name, i.start, i.end), i.ins[-1].get_name(), '[ CHILDS = ', ', '.join( "%x-%x-%s" % (j[0], j[1], j[2].get_name()) for j in i.childs ), ']', '[ FATHERS = ', ', '.join( j[2].get_name() for j in i.fathers ), ']', i.free_blocks_offsets

      for ins in i.get_ins() :
         print "\t\t %x" % idx, ins.get_name(), ins.get_operands()
         idx += ins.get_length()

      print ""

The most common attributes in a basic block are :

  • name : the name of the basic block (string)
  • start : a value which represents the offset where the basic block start (integer)
  • end : a value which represents the offset where the basic block finish (integer)
  • childs : a list of objects which represent the childs of the basic block
  • fathers : a list of objects which represent the fathers of the basic block
  • ins (or with get_ins method) : a list of Instructions objects of the basic block

Information

Packages

It's possible to view all used packages in your applications by using the tainted_packages field. With this field you can get packages, and every path where a package is used. With the access_flag() method of a package, you can know if a package is used in a call or during the creation of an object. 

for m, _ in x.tainted_packages.get_packages() :
   print "package : ", repr(m.get_info())
   for path in m.get_paths() :
       print "\t\t =>", path.get_access_flag(), path.get_method().get_class_name(), path.get_method().get_name(), path.get_method().get_descriptor(), path.get_bb().get_name(), "%x" % (path.get_bb().start + path.get_idx() )

It's possible to get all internal calls of the application : 

for j in x.tainted_packages.get_internal_packages() :
   print "\t %s %s %s %x ---> %s %s %s"  % (j.get_method().get_class_name(), j.get_method().get_name(), j.get_method().get_descriptor(), j.get_bb().start + j.get_idx(), j.get_class_name(), j.get_name(), j.get_descriptor())

or all external calls of the application : 

for j in x.tainted_packages.get_external_packages() :
   print "\t %s %s %s %x ---> %s %s %s" % (j.get_method().get_class_name(), j.get_method().get_name(), j.get_method().get_descriptor(), j.get_bb().start + j.get_idx(), j.get_class_name(), j.get_name(), j.get_descriptor())

You can search for a specific packages by using regular expressions : 

paths = x.tainted_packages.search_packages( "Landroid/telephony/" )
analysis.show_PathP( paths )

or search specific methods : 

paths = x.tainted_packages.search_methods( "Ljavax/crypto/", "generateSecret", ".")
analysis.show_PathP( paths )

Permissions

It's easy to get where a permission is used in an application : 

perms_access = x.get_permissions( [] )
for perm in perms_access :
   print "PERM : ", perm
   analysis.show_PathP( perms_access[ perm ] )

or with : 

show_Permissions( x )
